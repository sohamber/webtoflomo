# Webtoflomo

一个极简、高效的 Chrome 浏览器扩展，助你实现网页内容的“闪念捕捉”，一键发送到 [flomo 浮墨笔记](https://flomoapp.com)。

![Logo](icon.png)

## ✨ 功能特性

* **👆 划词悬浮**：选中网页文字后，自动出现悬浮按钮，点击即发。
* **✋ 拖拽交互**：悬浮按钮挡住视线？按住即可随意拖拽到屏幕任意位置。
* **🎯 智能跟随**：按钮会自动吸附在光标结束位置，无论选择多长内容，操作就在手边。
* **🖱️ 右键菜单**：支持传统的右键菜单“发送到 flomo”。
* **🔗 自动溯源**：自动抓取当前网页标题和链接，以 Markdown 格式附加在笔记末尾。
* **🔒 隐私安全**：API Key 仅存储在本地浏览器中，支持密码掩码显示。

## 🚀 安装指南

由于本项目尚未上架 Chrome Web Store，需要通过“开发者模式”安装：

1.  下载本仓库的代码（点击 Code -> Download ZIP 并解压）。
2.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/`。
3.  打开右上角的 **“开发者模式” (Developer mode)** 开关。
4.  点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**。
5.  选择你解压好的文件夹。

## ⚙️ 配置说明

1.  安装后，点击浏览器右上角的扩展图标（绿色小灯泡）。
2.  在弹出的窗口中配置你的 **Flomo API 链接**（可在 Flomo 设置 > API 及插件中获取）。
3.  (可选) 设置悬浮按钮的默认出现位置（如：正右侧、右上角等）。
